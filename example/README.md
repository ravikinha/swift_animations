# Swift Animations Example

This example app demonstrates all the features of the `swift_animations` package.

## Running the Example

```bash
cd example
flutter run
```

## Examples Included

- **Fade In** - Simple fade in animation
- **Scale** - Scale animation with repeat
- **Rotate** - Rotation animation
- **Bounce** - Bounce effect with scale
- **Pulse** - Pulse animation on icon
- **Slide In Top** - Slide in from top with fade
- **Slide In Left** - Slide in from left with fade
- **Complex Animation** - Multiple animations combined
- **Fade In Scale** - Combined fade and scale
- **Delayed Animation** - Animation with delay
- **Repeat Count** - Limited repeat count
- **Staggered Animations** - Multiple items with different delays
- **Slide X and Y** - Custom slide animations

## Features Demonstrated

- All animation types (fade, scale, rotate, slide)
- Special effects (bounce, pulse)
- Duration shorthand (`.s`, `.ms`)
- Delays
- Repeat options
- Complex animation combinations
- Staggered animations
